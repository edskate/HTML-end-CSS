HTML end CSS

Cascading Style Sheet, mais conhecido como CSS, é uma linguagem de estilos que, assim como o HTML,
também não é considerada uma linguagem de programação.
Ela é responsável por separar a parte estrutural da aplicação (que ficará nas mãos do HTML) da parte estética.
Para utilizar o CSS, usamos da seguinte sintaxe:

Exemplo:

seletor {
propriedade: valor; }

Exemplo 2

O seletor será o elemento que queremos estilizar (podendo ser uma tag, uma classe, um identificador...),
a propriedade será o que iremos alterar (como color, font-size, width...)
e o valor será de fato a alteração (como por exemplo: red para color, 18px para font-size e assim por diante).

Sendo assim, quando utilizamos uma tag HTML, como o que usamos de exemplo acima, podemos estilizá-la da seguinte forma:

p{
color: blue; }

Com isso, todos os parágrafos ficarão azuis.
